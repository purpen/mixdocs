=================
公用接口模块
=================

幻灯图列表
-----------------
获取某个推荐位幻灯图列表

接口说明
~~~~~~~~~~~~~~

* API接口请求地址：``/common/slides``
* API接口请求方法：``GET``

请求参数
~~~~~~~~~~~~~~~

===========  ========  =========  ========  ====================================
名称          类型      是否必须    默认值     描述说明
===========  ========  =========  ========  ====================================
spot         String    必需                  位置编号
page         Number    可选         1         当前页码
per_page     Number    可选         10        每页数量
===========  ========  =========  ========  ====================================

返回示例
~~~~~~~~~~~~~~~~

JSON数据格式:

.. code-block:: javascript

    {
      "data": {
        "slides": [
          {
            "description": "情人节",
            "image": "_uploads/photos/180104/d27aae678ce6652.jpg",
            "link": "http://www.mi.com",
            "rid": 7,
            "sort_order": 3,
            "status": true,
            "title": "情人节",
            "type": 1
          },
          ...
        ]
      },
      "status": {
        "code": 200,
        "message": "Ok all right."
      },
      "success": true
    }


搜索关键词历史
----------------------
获取历史搜索过的关键词列表

接口说明
~~~~~~~~~~~~~~

* API接口请求地址：``/search/history``
* API接口请求方法：``GET``

请求参数
~~~~~~~~~~~~~~~

===========  ========  =========  ========  ====================================
名称          类型      是否必须    默认值     描述说明
===========  ========  =========  ========  ====================================
page         Number    可选         1         当前页码
per_page     Number    可选         10        每页数量
===========  ========  =========  ========  ====================================

返回示例
~~~~~~~~~~~~~~~~

JSON数据格式:

.. code-block:: javascript

    {
        "data": {
            "count": 1,
            "next": null,
            "prev": null,
            "search_items": [
                {
                    "query_word": "电源",
                    "search_at": 1520149158,
                    "search_times": 1,
                    "total_count": 0
                }
            ]
        },
        "status": {
            "code": 200,
            "message": "Ok all right."
        },
        "success": true
    }
