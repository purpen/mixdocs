发布橱窗¶

接口说明¶

  • API接口请求地址：/shop_windows
  • API接口请求方法：POST
  • API接口用户授权：token

请求参数¶

名称 类型 是否必须 默认值 描述说明
title String 必须   标题
description String 必须   简介
product_items Array 必须   橱窗商品
keywords Array 可选   关键词

请求示例¶

{
    "title": "橱窗标题",
    "description": "橱窗描述内容",
    "product_items": [
        {
            "rid": "8037295684",
            "cover_id": 115173
        },
        {
            "rid": "8758496102",
            "cover_id": 115181
        },
        {
            "rid": "8160827453",
            "cover_id": 115185
        }
    ],
    "keywords": ["女孩", "原创饰品", "七夕"]
}

  • API接口请求方法：POST
  • API接口用户授权：token

请求参数¶

名称 类型 是否必须 默认值 描述说明
name String 必填   标签名
sort_order Number 可选 1 排序

返回示例¶

JSON数据格式:

{
    "data": {
        "name": "女孩",
        "rid": 46,
        "sort_order": 1
    },
    "status": {
        "code": 201,
        "message": "All created."
    },
    "success": true
}
