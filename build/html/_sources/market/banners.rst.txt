营销工具/广告模块
====================


生成推广图
----------------------
生成带有小程序码宣传图

接口说明
~~~~~~~~~~~~~~

* API接口请求地址：``/market/wxa_card``
* API接口请求方法：``POST``

请求参数
~~~~~~~~~~~~~~~

=============  ========  =========  ========  ====================================
名称            类型      是否必须    默认值     描述说明
=============  ========  =========  ========  ====================================
auth_app_id    String    必填                  小程序id
path           String    必填                  访问路径
rid            String    必填                  场景参数
=============  ========  =========  ========  ====================================

返回示例
~~~~~~~~~~~~~~~~

JSON数据格式:

.. code-block:: javascript

    {
        "data": {
            "wxa_image": "https://kg.erp.taihuoniao.com/20180314/Fk1vEAP_tIVwmPRfHRfl8jpn07CZ.png"
        },
        "status": {
            "code": 200,
            "message": "Ok all right."
        },
        "success": true
    }


生成小程序码
----------------------
生成不同大小的小程序码

接口说明
~~~~~~~~~~~~~~

* API接口请求地址：``/market/wxa_code``
* API接口请求方法：``POST``

请求参数
~~~~~~~~~~~~~~~

=============  ========  =========  ========  ====================================
名称            类型      是否必须    默认值     描述说明
=============  ========  =========  ========  ====================================
auth_app_id    String    必填                  小程序id
path           String    必填                  访问路径
rid            String    必填                  场景参数
width          Number    必填                  小程序码宽度
=============  ========  =========  ========  ====================================

返回示例
~~~~~~~~~~~~~~~~

JSON数据格式:

.. code-block:: javascript

    {
        "data": {
            "wxa_image": "https://kg.erp.taihuoniao.com/20180314/Fk1vEAP_tIVwmPRfHRfl8jpn07CZ.png"
        },
        "status": {
            "code": 200,
            "message": "Ok all right."
        },
        "success": true
    }
