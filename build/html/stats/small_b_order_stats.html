生活馆订单汇总统计

接口说明¶

  • API接口请求地址：/stats/life_orders_collect
  • API接口请求方法：GET
  • API接口用户授权：token

请求参数¶

名称 类型 是否必须 默认值 描述说明
store_rid String 必需   生活馆rid

返回示例¶

JSON数据格式:

{
    "data": {
        "all_count": 11,  // 总计成交订单
        "today_count": 0  // 今日成交
    },
    "status": {
        "code": 200,
        "message": "Ok all right."
    },
    "success": true
}

生活馆订单交易记录¶

生活馆订单交易记录

接口说明¶

  • API接口请求地址：/stats/life_orders/transactions
  • API接口请求方法：GET
  • API接口用户授权：token

请求参数¶

名称 类型 是否必须 默认值 描述说明
date_range String 必需   统计日期 ‘2018-01’ ‘week’:近7天 ‘month’:近一月
store_rid String 必需   生活馆rid
status Number 可选 0 0、全部 1、待结算 2、成功 3、退款
page Number 可选 1 当前页码
per_page Number 可选 10 每页数量

返回示例¶

JSON数据格式:

{
    "data": {
        "count": 2,
        "not_settlement_not_read": 0,  // 待结算未读数量
        "refund_not_read": 0,  // 退款未读数量
        "success_not_read": 0,  // 成功未读数量
        "transactions": [
            {
                "actual_amount": 3,  // 收入
                "order_rid": "D18081453268074",  // 订单号
                "payed_at": 1534232904,  // 时间
                "status": 1  // 状态 1、待结算 2、成功 3、退款
            },
        ]
    },
    "status": {
        "code": 200,
        "message": "Ok all right."
    },
    "success": true
}

  • API接口请求方法：GET
  • API接口用户授权：token

名称 类型 是否必须 默认值 描述说明
store_rid String 必需   生活馆rid
page Number 可选 1 当前页码
per_page Number 可选 10 每页数量

{
    "data": {
        "statements": {
            "2018": {
                "7": {
                    "statements": [
                        {
                            "actual_account_amount": 11,  // 实际提现金额
                            "actual_amount": 11,  // 提现金额

                        }

                    "total_amount": 11
                },
            }
        }
    },
    "status": {
        "code": 200,
        "message": "Ok all right."
    },
    "success": true
}
