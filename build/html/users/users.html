用户模块¶

获取用户最后进入的生活馆¶

核心小程序会用到, 移动端不需要

接口说明¶

  • API接口请求地址：/users/last_store
  • API接口请求方法：GET
  • API接口用户授权：token

JSON数据格式

请求 正确 返回结果：

{
    "data": {
        "last_store_rid": "78651792"
    },
    "status": {
        "code": 200,
        "message": "Ok all right."
    },
    "success": true
}
