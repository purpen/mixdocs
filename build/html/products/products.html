搜索商品列表¶

获取搜索商品列表

接口说明¶

  • API接口请求地址：/search/products
  • API接口请求方法：POST

请求参数¶

名称 类型 是否必须 默认值 描述说明
page Number 可选 1 当前页码
per_page Number 可选 10 每页数量
qk String 必选   搜索关键词
status Number 可选 -1 商品状态 0: 仓库中 1: 上架中 2: 下架中, 3: 已售罄

JSON数据格式:

{
    "data": {
        "next": false,
        "paginated_products": [
            {
                "commission_price": 30,
                "commission_rate": 5.3,
                "cover": "http://xxx/_uploads/photos/171010/fe7380064e18135.jpg",
